== Description ==

Sell many products in one Like Group Product. But you can use single price here.

## Features

1. Sell many products in one
2. Linked Unlimited products
3. Linked simple , variable or any type of products
4. Simple & easy to use !!

== Installation ==

    = Installation from within WordPress =

        1. Visit 'Plugins > 'Add New'.
        2. Search for 'Bulk Product Selling'.
        3. Install and activate the 'Bulk Product Selling' plugin.

    = Manual installation =

        1. Upload the entire `Bulk Product Selling` folder to the `/wp-content/plugins/` directory.
        2. Visit 'Plugins'.
        3. Activate the `Bulk Product Selling` plugin.
